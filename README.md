# Cantillon Smart Contracts

## Install packages:

```
npm i
```

## Running tests:

Create secrets.json like this

```
{
  "mnemonic": "your seed",
  "infuraApiKey": "your-infura-project-id3"
}
```

Configure node in test-environment.config.js

### Kovan

```
npm run test:kovan
```

### Kovan

```
npm run test:main
```
